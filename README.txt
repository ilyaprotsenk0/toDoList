"Create"

В правом верхнем углу расположена кнопка "Create", по нажатию появляется модальное окно которое содержит поле ввода для названия и поле ввода для описания (количество символов ограничено - 32 и 96 соответственно), а так же выпадающий список с выбором приоритета задачи. При нажатии вне модального окна оно скрывается.

В нижней части модального окна расположены кнопки "Confirm" и "Cancel"

	"Cancel"

	При нажатии модальное окно закрывается, введенные данные остаются.

	"Confirm"

	При нажатии закрывается модальное окно и выводится div элемент, который содержит все введенные данные. Задаче присваивается статус "Open".

	В нижней части div элемента имеются два выпадающих списка — слева "Priority", справа "Options".

	Если название было незаполненно, появляется предупреждение и модальное окно, где можно повторно ввести название и по желанию изменить другие данные.

"Priority"

В этом деактивированном списке выводится значение приоритета задачи из выпадающего списка модального окна

"Options" или "..."

Выпадающий список действий, который содержит следующие опции - "Done", "Edit", "Delete".

	"Done"

	При выборе этой опции задаче присваивается статус "Done", фон блока окрашивается в зеленый цвет.

	"Delete"

	При выборе этой опции задача удаляется.

	"Edit"

	При выборе можно отредактировать название, описание задачи. Вместо выбора опций появляется кнопка "Apply", а в списке "Priority" теперь можно выбирать значения.

		"Apply"

		Собирает все полученные в режиме редактирования данные и выводит готовый блок.

		"Priority"

		Список, в котором можно изменить приоритет задачи. Содержит опции - "High", "Medium", "Low".

После успешного создания/редактирования/удаления задач, можно рассмотреть поиск по названию и фильтрацию по приоритету и статусу задачи. "Search", "Priority filter", "Status filter" находятся в хэдере сайта.

"Search"

Поле ввода, в котором при вводе символов выполняется поиск только по названию среди всех существующих задач. Для большего удобства регистр букв не учитывается.

"Priority filter" или "Priority"

Выпадающий список, который содержит следующие опции - "Priority", "High", "Middle", "Low". При выборе опций "High", "Middle" или "Low" скрываются блоки, которые не содержат такого типа приоритет. При выборе опции "Priority" выводятся все существующие блоки.

"Status filter" или "All"

Выпадающий список, который содержит следующие опции - "Open", "Done". При выборе любой из опций скрываются блоки, которые не содержат такого типа статус. При выборе опции "All" выводятся все существующие блоки.

Взаимодействие "Priority filter" и "Status filter"

Есть возможность одновременно выбрать и фильтрацию по приоритету, и фильтрацию по статусу. Порядок выбора не важен. Отобразится результат по выбранным значениям. Для дальнейшей корректной работы фильтров их следует переключить в "Priority" и "All" и применить фильтрацию снова.
